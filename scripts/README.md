# Scripts Directory

Automation and maintenance scripts for Tobb Sunil.

## Included Scripts

- `morning_briefing.py` - Daily summary of emails, calendar, weather
- `spark_health.py` - Health checks for Spark1 and Spark2
- `nightly_backup.sh` - Automatic nightly backups
- `archive_memory.py` - Archive old memory files

## Running Scripts

Scripts can be run manually or via cron jobs.

## Configuration

Most scripts look for configuration in `~/.config/openclaw/` or use environment variables.
