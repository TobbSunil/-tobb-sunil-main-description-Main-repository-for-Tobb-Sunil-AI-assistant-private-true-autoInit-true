# Daily Logs

Raw, unfiltered logs of what happens each day.

## Format

- `YYYY-MM-DD.md` - Daily notes with what happened, decisions made, insights gained

## Purpose

- Track daily activities and observations
- Capture raw data for later curation into MEMORY.md
- Provide context for specific time periods

## Retention

- 7 days: Full content
- Older: Archival or deletion
