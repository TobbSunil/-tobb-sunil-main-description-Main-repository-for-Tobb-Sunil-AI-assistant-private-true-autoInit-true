# Memory Curation Guidelines

This document contains distilled insights from daily logs.

## Format

- Curated, edited, and organized version of daily logs
- Contains important decisions, lessons learned, key insights

## Usage

- Reference for long-term memory
- Context for understanding past decisions
- Avoids duplicates from daily logs

## Maintenance

- Updated during memory garbage collection (every 3 days)
- Blends similar insights
- Removes outdated or redundant information

## Security

- Contains personal context
- Should not be shared publicly
- Only loaded in main sessions (not shared contexts)
